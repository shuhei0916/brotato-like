# brotato-like
## 概要
brotato like なゲームをgodotで開発してみます

## プロジェクトが一時停滞していた理由: 
- state状態遷移の重要性を認識師ながらも、ドキュメントを読むのが億劫だったため（https://gameprogrammingpatterns.com/state.html）。
	- 対策：読むのが面倒なドキュメントはAIにドキュメントをまとめてもらい、概要をつかむ。
- 起こっている不具合を正しく認知出来ていない。認知できても正しく言語化出来ない。
	- 対策：経験を積む、
- 目標意識が曖昧になった。
	- 対策：目標をまとめたドキュメントや、コンセプトアートの作成
